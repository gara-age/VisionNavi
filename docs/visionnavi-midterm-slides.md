# VisionNavi 중간발표 슬라이드 구성안

## 슬라이드 1. 제목 / 한 줄 소개

### 핵심 메시지

- VisionNavi는 고령자를 위한 음성 기반 웹/데스크톱 자동화 도우미입니다.
- 목표는 "명령 이해"를 넘어서 "실제 작업 수행"까지 연결하는 것입니다.

### 넣을 시각 자료

- VisionNavi 로고
- 메인 화면 UI 이미지 1장

### 발표 포인트

VisionNavi는 한국과 일본의 고령자를 주요 사용자로 두고 있습니다.  
음성이나 자연어 텍스트로 요청하면 웹 검색이나 메모 작성 같은 작업을 대신 수행하는 시스템입니다.

---

## 슬라이드 2. 문제 정의

### 핵심 메시지

- 기존 웹/PC 사용은 고령자에게 여전히 복잡합니다.
- 기존 자동화는 정해진 step에는 강하지만, 화면 변화와 예외 상황에는 약합니다.

### 넣을 시각 자료

- 복잡한 웹 탐색 또는 여러 단계 UI를 보여주는 예시 이미지
- "기존 step 기반 자동화의 한계"를 간단히 정리한 도식

### 발표 포인트

검색어를 입력하고, 결과를 보고, 버튼을 누르고, 다음 화면으로 이동하는 과정 자체가 고령자에게 부담입니다.  
기존 자동화는 미리 정해진 절차를 반복하는 데는 좋지만, 실제 화면이 달라지면 쉽게 깨진다는 문제가 있었습니다.

---

## 슬라이드 3. VisionNavi 해결 방식

### 핵심 메시지

- VisionNavi는 입력 해석부터 실행, 검증, 안내까지 하나의 흐름으로 관리합니다.
- 단순 executor가 아니라 orchestrator 구조를 사용합니다.

### 넣을 시각 자료

- `입력 -> 명령 이해 -> 실행 채널 선택 -> 실제 조작 -> 결과 안내` 흐름도

### 발표 포인트

사용자가 말하거나 입력하면, VisionNavi가 먼저 요청을 정리하고 웹 작업인지 데스크톱 작업인지 판단합니다.  
그 다음 적절한 실행 채널을 골라 실제 브라우저나 Windows 앱을 조작하고, 결과를 다시 사용자에게 안내합니다.

---

## 슬라이드 4. 현재 구현 현황

### 핵심 메시지

- 검색/읽기, 메모장 작성, 음성 입력, wakeword, 고령자 친화 UI까지 중간 단계 구현을 진행했습니다.
- 모든 기능이 완성형은 아니지만 실제 동작하는 시나리오가 확보되어 있습니다.

### 넣을 시각 자료

- 기능 현황 표
- 가능하면 `완료 / 진행중` 형태의 간단한 상태 표

### 발표 포인트

현재는 검색 및 읽기, 메모장 작성, STT, wakeword 실험, UI 개편이 주요 축입니다.  
특히 검색/읽기와 메모장 작성은 실제 시연 가능한 수준으로 발전해 있고, 길찾기 시나리오는 아직 internal baseline 비중이 큽니다.

---

## 슬라이드 5. 시연 포인트

### 핵심 메시지

- VisionNavi는 명령을 이해하는 데서 끝나지 않고 실제 화면 작업으로 이어집니다.
- 시연에서는 입력, 실행, 결과 안내의 전체 흐름을 보여줍니다.

### 넣을 시각 자료

- 시연 영상 캡처 2~3장
- 예: 검색 자동화, 메모장 작성, 팝업 결과 안내

### 발표 포인트

이번 시연에서는 음성 또는 텍스트 입력을 받아 실제 검색을 수행하거나 메모장을 조작하는 모습을 보여줄 수 있습니다.  
이 과정에서 단순 응답형 챗봇이 아니라 작업 실행형 시스템이라는 점이 드러납니다.

---

## 슬라이드 6. 현재 한계와 다음 단계

### 핵심 메시지

- 지금 가장 중요한 것은 모델 교체 자체보다 실행 구조의 안정화입니다.
- provider, query, language를 더 강하게 고정하고 external agent 안정성을 높이는 것이 다음 과제입니다.

### 넣을 시각 자료

- `현재 한계 -> 다음 개선 방향` 2열 정리

### 발표 포인트

현재는 일부 명령에서 검색어가 변형되거나 다른 검색 엔진으로 이탈하는 문제가 있습니다.  
그래서 앞으로는 constraint/validator를 강화하고 external browser 및 desktop agent의 안정성을 높이는 방향으로 개발할 계획입니다.

---

## 슬라이드 7. 마무리

### 핵심 메시지

- VisionNavi는 고령자의 디지털 접근성을 높이기 위한 실사용형 자동화 프로젝트입니다.
- 현재는 중간 단계 구현을 확보했고, 앞으로는 agent 완성도와 안정성을 높여갈 예정입니다.

### 넣을 시각 자료

- 메인 UI 이미지
- 핵심 문장 3줄

### 발표 포인트

VisionNavi는 단순한 기술 데모가 아니라, 실제로 고령자의 웹과 PC 사용을 돕는 도구를 목표로 합니다.  
현재는 핵심 시나리오를 구현한 중간 단계이며, 이후에는 더 안정적이고 범용적인 에이전트형 시스템으로 발전시키고자 합니다.
